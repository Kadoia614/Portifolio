# Portfólio - Miguel Moraes

Portfólio pessoal desenvolvido para apresentar a trajetória profissional, formação, competências técnicas e projetos de Miguel Moraes, desenvolvedor Front-end, Fullstack e UX/UI Designer localizado em São Paulo - SP.

## Sobre o projeto

O site é uma landing page estática, construída com HTML e CSS puro, com foco em:

- apresentação profissional;
- experiência de trabalho;
- formação acadêmica;
- habilidades técnicas;
- cursos e competências complementares;
- idiomas;
- projetos publicados;
- contato profissional.

## Preview

O projeto possui uma página principal em `index.html`, com imagens de apoio na pasta `img/` e estilos organizados em módulos dentro da pasta `css/`.

## Como executar o projeto

Como o projeto é estático, não é necessário instalar dependências.

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd portfolio
```

3. Abra o arquivo `index.html` no navegador.

Também é possível executar com a extensão Live Server no VS Code para facilitar o desenvolvimento local.

## Como publicar

O projeto pode ser publicado facilmente no GitHub Pages:

1. Envie o repositório para o GitHub.
2. Acesse as configurações do repositório.
3. Vá até `Pages`.
4. Selecione a branch principal.
5. Publique a partir da raiz do projeto.
